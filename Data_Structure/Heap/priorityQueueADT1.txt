//우선순위 큐 ADT
*객체: 우선순위를 가진 n개의 element형 요소들의 모임

function create() do
    우선순위 큐를 생성한다.
end

function init(q) do
    우선순위 큐 q를 초기화한다.
end

function is_empty(q) do
    우선순위 큐 q가 비어 있는지 검사한다.
end

function is_full(q) do
    우선순위 큐 q가 가득 찼는지 검사한다.
end

function insert(q,x) do
    우선순위 큐 q에 요소 x를 추가한다.
end

function delete(q) do
    우선순위 큐로부터 가장 우선순위가 높은 요소를 삭제하고 이 요소를 반환한다.
end
//우선순위 높은 애부터 delete해서 반환하가에 delete가 중요함

function find(q) do
    우선순위가 가장 높은 요소를 반환한다.
end

우선순위 큐: 큐에서 들어온 순서와 무관하게 우선순위가 높은 데이터부터 나가는 큐임.
이를 힙으로 구현하고자 하는 것임.
우선순위를 부여함에 따라 FIFO 큐나 LIFO 스택까지 포괄(구현) 가능함
(먼저 들어온/나중에 들어온 순서에 따라 우선순위 부여하는 방식으로)

(정렬/순서 없는)배열에 넣을 수도 있고 (정렬/순서 없는)연결 리스트에 넣을 수도 있고 구현 방식은 다양함.
트리 구조인 힙이 삽입/삭제 연산에서 가장 효율적인 자료 구조
